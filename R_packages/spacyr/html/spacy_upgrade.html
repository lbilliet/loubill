spacy_upgrade {spacyr}R Documentation

Install spaCy in conda environment

Description

Install spaCy in conda environment

Usage

spacy_upgrade(conda = "auto", envname = "spacy_condaenv",
  lang_models = "en")

Arguments

conda

Path to conda executable. Default "auto" which automatically find the path

envname

character; name of conda environment to upgrate spaCy

lang_models

Language models to be upgraded. Default NULL (No upgrade). A vector of multiple model names can be used (e.g. c("en", "de"))
