nearest_to {wordVectors}R Documentation

Nearest vectors to a word

Description

This a wrapper around closest_to, included for back-compatibility. Use closest_to for new applications.

Usage

nearest_to(...)

Arguments

...

See 'closest_to'

Value

a names vector of cosine similarities. See 'nearest_to' for more details.

Examples


# Recommended usage in 1.0:
nearest_to(demo_vectors, demo_vectors[["good"]])

# Recommended usage in 2.0:
demo_vectors %>% closest_to("good")
