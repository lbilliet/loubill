restaurant {easyGgplot2}R Documentation

Restaurant bill for man and woman

Description

Total bill at different time

Usage

data(restaurant)

Format

A data frame with 4 observations on the following 3 variables.

sex

a factor with levels Female Male

time

a factor with levels Lunch Dinner

total_bill

a numeric vector

References

http://www.sthda.com

Examples

data(restaurant)
ggplot2.barplot(data=restaurant, xName="sex", yName='total_bill', width=0.5)
